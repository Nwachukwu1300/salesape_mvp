This frontend uses Vite + React (previously Next.js references removed).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open the dev server URL printed by `vite` (typically http://localhost:3000).

Start editing the app in `src/` — Vite will hot-reload changes.

This project uses Vite and standard web font loading; remove Next-specific font optimizations.

## Learn More

For info on Vite, see the Vite docs: https://vitejs.dev/guide/

## Deploy
Build with `vite build` and deploy the `build/` output to your static host of choice.
